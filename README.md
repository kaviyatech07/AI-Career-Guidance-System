# 🤖 AI Career Guidance System

An AI-powered Career Guidance System that recommends suitable career paths based on a user's skills and interests. Built with Python and Streamlit.

---

## ✨ Features

- Career prediction based on skills
- Personalized learning roadmap
- Simple and interactive UI
- Easy to use
- Beginner-friendly

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Git
- GitHub

## 🚀 Installation

```bash
git clone https://github.com/YOUR_USERNAME/AI-Career-Guidance-System.git
```

```bash
cd AI-Career-Guidance-System
```

```bash
pip install -r requirements.txt
```

```bash
streamlit run app.py
```
